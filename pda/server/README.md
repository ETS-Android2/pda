Instalação:
- Instalar Node.js/npm;
- Instalar Memurai (Redis para Windows):
    - Configurar para a porta 3035.
- Instalar pm2 (npm install -g pm2);
- Criar uma variável de sistema PM2_HOME com o atributo c:\etc\.pm2;
